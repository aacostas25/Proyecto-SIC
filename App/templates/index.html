Proyecto Sistemas Inteligentes Computacionales

El proyecto fue creado con el fin de suministrar una herramienta en la cual se pueda visualizar la implementación de Sistemas inteligentes, mas precisamente Redes Neuronales en al menos 2 casos diferentes. La implementación del proyecto consta de 3 bases de datos conocidas y una zona de playground donde el usuario puede crear datos sintéticos para hacer mas directa la interacción entre los datos de entrada y el aprendizaje de la Red Neuronal.
